1. MIGRAZIONE SEO RENDERING — PRIORITÀ MASSIMA
==================================================

PROBLEMA ATTUALE:
Il sito è una React SPA CSR pura con metadata iniettati via useEffect.
Google riceve inizialmente:
<div id="root"></div>

Questo crea:
- second wave indexing
- lentezza indicizzazione
- rischio pagine non renderizzate
- problemi Bing/social previews
- scarso trust SEO

IMPLEMENTARE UNA DELLE SEGUENTI SOLUZIONI:

PREFERITA:
- SSR o SSG compatibile con Vite

ALTERNATIVE ACCETTABILI:
- vite-ssg
- react-snap
- prerender.io
- prerender middleware
- static prerendering delle pagine SEO

OBIETTIVO:
Googlebot deve ricevere HTML COMPLETO già renderizzato con:
- title
- meta description
- canonical
- JSON-LD
- H1/H2
- contenuto SEO
- internal links

senza dover eseguire JavaScript.

IMPORTANTE:
- Le pagine dashboard/auth possono restare CSR
- SOLO le pagine SEO devono essere prerenderizzate/SSR
