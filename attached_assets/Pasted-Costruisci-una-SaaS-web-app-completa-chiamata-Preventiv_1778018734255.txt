Costruisci una SaaS web app completa chiamata "PreventivoAI".

L’obiettivo è creare un sistema AI per generare preventivi professionali PDF per freelance, artigiani e piccole imprese italiane.

STACK TECNICO:
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Supabase (auth + database)
- Stripe (pagamenti subscription + one-shot)
- OpenAI API (AI generazione preventivi)
- Puppeteer (generazione PDF da HTML)

---

# 🎯 OBIETTIVO DEL PRODOTTO

L’utente inserisce una descrizione in linguaggio naturale del lavoro da preventivare.
L’AI trasforma il testo in un preventivo strutturato (righe, quantità, prezzi, IVA, totale) e genera un PDF professionale.

Il download del PDF è bloccato da paywall.

# 🤖 LOGICA AI (FONDAMENTALE)

Usare OpenAI API per trasformare input testuale in JSON strutturato.

### Prompt interno AI (IMPORTANTISSIMO):

Sei un assistente esperto di contabilità e preventivi italiani.

Ricevi una descrizione libera di un lavoro.

Devi restituire SOLO JSON valido con:

- cliente (se presente)
- descrizione lavori
- lista righe preventivo:
  - descrizione
  - quantità
  - unità (mq, ore, pezzi)
  - prezzo unitario
  - totale riga
- subtotal
- IVA %
- totale finale

Regole:
- usa prezzi realistici per mercato italiano
- se manca info, fai assunzioni plausibili
- non scrivere testo fuori dal JSON
